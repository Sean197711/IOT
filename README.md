# 物联网 B 端微信小程序 UI 组件库

本组件库是为物联网行业 B 端业务场景定制的 **微信小程序原生组件库**，旨在提升设计一致性和开发效率。

## 特性
- 🎯 面向 B 端业务，高信息密度 & 状态可视化
- 📱 原生小程序开发，轻量无依赖
- 🎨 与设计规范深度绑定，保持一致性
- ⚡ 可复用、可扩展、可维护

## 快速开始
```bash
# 克隆代码
git clone https://git.xxx.com/iot-miniapp/ui-kit.git

# 安装依赖
npm install

# 在小程序中引入
import 'ui-kit/dist/styles.wxss'
```
